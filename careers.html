Software Engineer Intern

Description:

Interns will have the opportunity to work alongside the technical co-founders on a full-stack application with a logistics use case. The role is primarily front-end/interface focused, but there are ample opportunities to work on the back-end/database/cloud infrastructure, as well as mobile development.
Interns will have ownership over software features. They will be given impactful projects to work on and will be collaborating closely with the co-founders who will be working on related deliverables. Although the work will be largely independent in nature, there will be ample support and mentorship to ensure that interns are learning and able to contribute.
You will get out of this internship what you put into it. We believe it is in our and your best interest for you to succeed and make impactful contributions.

By the end of the internship, students will have learned:

  • Web development and how to build an application in the cloud
  • Best practices for coding and design
  • Project management skills (how to scope things out, etc)
  • New technologies and languages

We are looking for:

  • A willingness to learn new things and a desire to succeed and self-improve
  • A passion for software development
  • Someone who likes having ownership over their projects and can work independently
  • An interest in entrepreneurship

Required skills:

  • Experience with HTML, CSS and Javascript are preferred but not required
  • Experience with React.js, or any frontend Javascript framework is preferred but not required
  • Experience with Terraform, AWS, docker is an asset
  • Experience with Python is an asset
  • Experience with mobile development (IOS/Android) is an asset
  • General web development and API experience is an asset

Tech Stack:

  • React JS frontend with Recoil, React Router and Material UI
  • Python/Flask backend
  • Postgres
  • AWS
  • Docker
  • Terraform
  • Git
  • JIRA
This is a REMOTE position. We will be working out of Waterloo, and there will be opportunities to meet and work together if you are also in the Waterloo region.
